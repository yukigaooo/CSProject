FACTORING

For the easy case of factoring, you will find two numbers that will not only multiply to equal the constant term "c", but also add up to equal "b".For example :X^2-3x+2 can be complete factoring the answer is (x-1) (x-2), but this is not the final answer, you need to set both factor as x-1=0 and x-2=0, and then you can find out two x's answer, which is x=1 and x=2. However, some equation can not be solved this way,a formula is used to solve it. Answers always have 2 solutions. They can be either rational or irrational. However, in some cases the equation can only get one solution because the solutions are the same. Example question : solve for X^2+4x+3=0 Answer : (x+3) (x+1) x=-3 x=-1

FUNCTIONS

A function relates an input to an output. f(x) = ... " is the classic way of writing a function. Basically a function is that the same x value can not have more than 1 y value. Moreover, for function, in one x value you can not have more than one value. For example {(1,0),(2,0),(3,1)} could be a function. But {(1,0),(1,1), (1,2)} can not be a function because the x value which is one has more than 1 y value .

SLOPE

Equation y=mx+b where b is the y intercept that is attached to the the x-axis , m is the slope for the equation. If m is negative the slope will go down, if n is possible it will considering a possible line. Example : y=2x+3 and y=2x-3 are all possible slope. How to calculate slope? as shown on the picture, if you want to find a slope of the line you need to use the equation y2-y1/x2-x1 so for this line the equation will be ⅔ x and the y intercept is where the line intercept with the y axis. Which is 0.5. So we can determine the line as y=2/3x+0.5 STEPS Step 1: Put the equation in Slope Intercept Form Step 2: Graph the y-intercept point (the number in the b position) on the y-axis. Step 3: From the point plotted on the y-axis, use the slope to find your second point. Remember, the slope is the number in the m position in your equation. Step 4: Draw your line using the two points you plotted (y-intercept (b) first, slope (m) second. Be sure your line is pointing the right way.
